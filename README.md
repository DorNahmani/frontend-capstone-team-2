השוליה של האלכימאי (The Alchemist's Apprentice) 🧪✨
משחק Escape Room אינטראקטיבי בדפדפן המבוסס על חידות לוגיות, סאונד ואווירה קסומה.

📖 תיאור המערכת
במשחק זה, השחקן נכנס לנעליו של שוליית אלכימאי שנעלם במפתיע. עליו לעבור דרך 5 תחנות שונות במעבדה המסתורית, לפתור חידות קריסטלים, לרקוח שיקויים ולצוד רוחות רפאים כדי לגלות את סודות המאסטר ולצאת לחופשי. הפרויקט מנהל מצבי התקדמות (States) בין דפים שונים כדי להבטיח חוויה ליניארית ומאתגרת.

🛠️ טכנולוגיות בשימוש
Frontend: HTML5, CSS3 (Flexbox & Grid).

Logic: Vanilla JavaScript (ES6+).

Storage: LocalStorage API לשמירת התקדמות השחקן וניהול ה-State ללא צורך בשרת.

Audio: Web Audio API לשילוב מוזיקת רקע דינמית ואפקטים קוליים המגיבים לפעולות השחקן.

Deployment: GitHub Pages.

🌟 סיכום פיצ'רים (Features)
מערכת ניהול משתמש: קליטת שם השחקן בתחנה הראשונה והצגתו כהודעת ברוך הבא אישית לאורך כל שלבי המשחק.

בקרת התקדמות: מנגנון מובנה המונע מהשחקן "לדלג" לתחנות מתקדמות בלי לפתור את החידות הקודמות (מבוסס על ערך ה-Status בזיכרון הדפדפן).

אינטראקטיביות דינמית: חידות המבוססות על שינוי צבעים בזמן אמת, אנימציות CSS (כמו רעידות הדרקון) ותגובות מיידיות ללחיצות.

חוויה קולית מלאה: שילוב אפקטים קוליים (שאגות, לחיצות קסומות) ומוזיקת רקע שמתחילה אוטומטית לאחר אינטראקציה ראשונה.

עיצוב רספונסיבי: התאמה מלאה למכשירים ניידים (Mobile Friendly) באמצעות Media Queries.
