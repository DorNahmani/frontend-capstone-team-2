השוליה של האלכימאי – The Alchemist's Apprentice 🧪✨
ברוכים הבאים למעבדה המסתורית! "השוליה של האלכימאי" הוא משחק Escape Room אינטראקטיבי המתרחש בדפדפן. אתם נכנסים לנעליו של שוליה צעיר המגלה שהאלכימאי הגדול נעלם בנסיבות מחשידות. כדי לצאת מהמעבדה בשלום, עליכם לעבור בין חמש תחנות שונות, לפתור חידות קסומות ולהשתמש בתושייה שלכם.

📖 על המשחק
המשחק בנוי כחוויה ליניארית המשלבת היגיון, מהירות תגובה ואווירה. בכל שלב במעבדה תפגשו אתגר אחר: מהתעסקות עם קריסטלים עתיקים ועד לרקיחת שיקויים מסוכנים. כדי להשלים את המשימה, המערכת מלווה אתכם באפקטים קוליים ומוזיקה שמשתנה בהתאם להתקדמות שלכם – רק היזהרו לא להעיר את הדרקון!

🛠️ הצד הטכני (מה קורה מאחורי הקלעים)
הפרויקט נבנה בדגש על ביצועים גבוהים וחוויית משתמש חלקה, תוך שימוש בטכנולוגיות הבאות:

ממשק משתמש: נעשה שימוש ב-HTML5 ו-CSS3 מתקדם (Flexbox ו-Grid) כדי להבטיח שהמעבדה תיראה טוב גם במחשב וגם בטלפון הנייד.

לוגיקה: המשחק מבוסס על Vanilla JavaScript בלבד, ללא ספריות חיצוניות, מה שמאפשר טעינה מהירה במיוחד.

ניהול נתונים: השתמשנו ב-LocalStorage כדי לשמור את השם שלכם ואת הסטטוס במשחק. זה מאפשר לכם לחזור בדיוק לנקודה שבה הפסקתם גם אם רעננתם את הדף.

חוויה קולית: שילבנו את ה-Web Audio API כדי ליצור סביבה קולית דינמית שמגיבה לפעולות שלכם בזמן אמת.

🌟 פיצ'רים בולטים
התאמה אישית: המערכת זוכרת את השם שלכם ומלווה אתכם באופן אישי לאורך כל התחנות.

בקרת שלבים: המשחק כולל מנגנון חכם שמוודא שפתרתם את החידה הנוכחית לפני שתוכלו להתקדם לתחנה הבאה.

אינטראקטיביות: לחיצה על אלמנטים גורמת לשינויים ויזואליים מיידיים, אנימציות CSS ואפקטים קוליים.

רספונסיביות: המשחק הותאם במיוחד למובייל כדי שתוכלו לפתור חידות גם בדרך.
